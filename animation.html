animation

This topic introduced the programming language javascript, which Is and indispensable part og the modern internet and is used to code advanced and dynamic websites as well as create interactive elements such as burger menus, slideshows, snap scrolls etc. Particular emphasis was placed on working professionally with the design of graphic elements.

The game design was analysed and view on how it engages users through theory and conventions within UI-elements, figure elements, background design and composition techniques. The game design is developed according to the modern conventions and draws the graphics for the game in Adobe Illustrator.

In additions and introductions to the work with activity and statemachine diagrams were made. These are used to plan the more complicated interactive flow. Javascript was used together with css animations, to create a small game. The principles of css and javaScript can be continued to develop more advanced websites.
